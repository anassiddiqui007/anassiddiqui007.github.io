Predicting colour of organic dyes in solution using theory and simulation

TDDFT can be used for generating absorption spectrum of a solute in solvent, and hence identifying the colour it would exhibit in the solution. My project involves examining the current methodology and improving its accuracy and computational cost. The whole analysis is performed using AMBER Molecular Dynamics, NWChem sofware package as well as Python. Up till now, I have already designed several original algorithms for finding and arranging solvent molecules around solute, for input file generations as well as for absorption spectra generation.

Phase Field modelling for microstructure evolution

In this project, I worked with a C/C++ framework MOOSE to study microstructure evolution in 2D as well as 3D. I wrote my own sophisticated codes for emulating the experimental conditions for the microstructures and analyzed their changes over time. Some examples over which I worked included Allen-Cahn, Cahn-Hilliard and spinodal decomposition of iron-chromium alloy.

Relative computations of a Raspberry Pi cluster

For this project, I had to assemble a cluster of Raspberry Pi microcontrollers with minimal documentation and configure them for parallel execution of codes. I had to independently create Linux shell files for establishing a communication channel between the devices. Following this, I had to modify programming codes to run parallelly on the cluster and then benchmark the computational cost with different cluster sizes as well as the modern PC. The assembled cluster can be used for several small scientific calculations which requrire parallel execution.
